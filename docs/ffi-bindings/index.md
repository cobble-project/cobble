# FFI Bindings

Cobble's core engine is written in Rust, but its functionality is accessible from other languages through foreign function interface (FFI) bindings.

| Binding | Language | Crate |
|---------|----------|-------|
| [Java](java) | JVM (Java, Kotlin, Scala, …) | `cobble-java` |
